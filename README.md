# SimpleSNS - SNS学習アプリ

1つのHTMLファイルで完結するシンプルなSNSアプリです。SNSの基本的な仕組みを学習できます。

## 🚀 特徴

- **ユーザー登録・ログイン**: ローカルストレージを使用した認証システム
- **投稿機能**: 280文字以内のテキスト投稿
- **タイムライン**: 投稿を時系列で表示
- **いいね機能**: 投稿にいいねを付ける
- **レスポンシブデザイン**: PC・スマートフォン対応

## 📋 使い方

### 1. アプリケーションの起動
`index.html` をダブルクリックしてブラウザで開いてください。

### 2. ユーザー登録
- 「新規登録」をクリック
- ユーザー名（3文字以上）とパスワード（6文字以上）を入力
- 「新規登録」ボタンをクリック

### 3. ログイン
- ユーザー名とパスワードを入力
- 「ログイン」ボタンをクリック

### 4. 投稿
- テキストエリアに投稿内容を入力（280文字以内）
- 「投稿」ボタンをクリック
- またはCtrl+Enterで投稿

### 5. いいね
- 各投稿の❤️ボタンをクリックしていいねを付ける

## 🛠️ 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **データ保存**: localStorage（ブラウザのローカルストレージ）
- **レスポンシブ**: CSS Grid, Flexbox

- Firefox

- Edge

## 🎯 学習ポイント

このアプリケーションで学べるSNSの基本概念：

1. **ユーザー認証**: 登録・ログイン・ログアウト
2. **データ管理**: ユーザー情報・投稿データの管理
3. **UI/UX**: SNSらしいインターフェース設計
4. **リアルタイム更新**: いいねやタイムライン表示
5. **バリデーション**: 入力チェック・エラーハンドリング

## 🔧 カスタマイズ


### デザインの変更
`<style>` タグ内のCSSを編集してデザインをカスタマイズできます。

## 📝 データ構造

### ユーザーデータ
```javascript
{
  username: "ユーザー名",
  password: "パスワード",
  createdAt: "作成日時"
}
```

### 投稿データ
```javascript
{
  id: 投稿ID,
  username: "投稿者名",
  content: "投稿内容",
  timestamp: "投稿日時",
  likes: ["いいねしたユーザー名の配列"]
}
```
